## はじめに

AWS発のAI搭載IDE「Kiro」を使っていて、**マウス操作が面倒**と感じたことはありませんか？

特にAIチャットとエディタを行き来する場面では、キーボードだけで操作できると作業効率が格段に上がります。

この記事では、Kiro IDEで使える**実践的なキーボードショートカット**をまとめました。

## Kiro IDEとは

KiroはAWSが2025年にリリースした**AI搭載の開発環境**です。VS Codeをベースにしており、Claude AIとの対話機能が統合されています。

- VS Codeベースなので、既存のショートカットがほぼそのまま使える
- AIチャット機能が組み込まれている
- スペック駆動開発をサポート

## 最重要ショートカット3つ

まずはこの3つを覚えれば、マウスなしで基本操作ができます。

| 操作 | ショートカット | 用途 |
|------|---------------|------|
| **チャットにフォーカス** | `Ctrl + L` | AIに質問したい時 |
| **エディタに戻る** | `Ctrl + 1` | コード編集に戻る |
| **ターミナル切替** | `` Ctrl + ` `` | コマンド実行 |

## チャット操作

Kiro最大の特徴であるAIチャット関連のショートカットです。

| 操作 | ショートカット |
|------|---------------|
| チャットパネルを開く/フォーカス | `Ctrl + L` |
| コマンドパレット | `Ctrl + Shift + P` |
| 選択範囲をチャットに送る | 選択後 `Ctrl + L` |

:::note info
`Ctrl + L`は最も使用頻度が高いショートカットです。エディタで作業中、すぐにAIに質問できます。
:::

複数のエディタやパネルを行き来する際に使います。

## 実践的なワークフロー

### パターン1: コードを書いてAIにレビューしてもらう

1. `Ctrl + 1` でエディタにフォーカス
2. コードを書く
3. レビューしたい範囲を選択
4. `Ctrl + L` でチャットにフォーカス
5. 「このコードをレビューして」と入力

### パターン2: AIの回答を見ながらコード修正

1. `Ctrl + L` でAIに質問
2. 回答を確認
3. `Ctrl + 1` でエディタに戻る
4. 修正を反映
5. 繰り返し

### パターン3: ターミナルでコマンド実行→結果をAIに共有

1. `` Ctrl + ` `` でターミナルを開く
2. コマンド実行
3. エラーが出たら選択してコピー
4. `Ctrl + L` でチャットへ
5. 「このエラーを解決して」

## まとめ

最低限覚えるべきショートカット：

```
Ctrl + L     → チャット
Ctrl + 1     → エディタ
Ctrl + `     → ターミナル
Ctrl + P     → ファイル検索
Ctrl + Shift + P → コマンドパレット
```

これだけでマウスに手を伸ばす回数が激減します。

Kiro IDEはVS Codeベースなので、VS Codeのショートカットがほぼそのまま使えます。既にVS Codeに慣れている方は、`Ctrl + L`（チャット）だけ覚えれば十分です。
